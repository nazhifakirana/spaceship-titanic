# Spaceship Titanic ML Project

## Description
This project builds a Logistic Regression model using an end-to-end sklearn Pipeline to predict whether a passenger is transported.

## Features
- Feature Engineering inside Pipeline
- ColumnTransformer for preprocessing
- Logistic Regression model
- Streamlit app for deployment

## How to Run

Train the model:
python src/pipeline.py

Run the app:
streamlit run app.py
